# Behavioral-Responses-to-State-Level-Vaping-Flavor-Bans
This project evaluates the impact of state-level ENDS flavor bans on vaping and substance use using BRFSS data (2016–2023). Methods include DiD, event study, logit models, and double-selection Lasso. Results show reduced vaping but substitution toward combustible tobacco, with heterogeneity by mental health status.
